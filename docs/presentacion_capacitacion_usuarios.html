Capacitacion de usuarios

SERVIU Subsidios

Sistema de gestion de subsidios habitacionales para control de solicitantes, comites, solicitudes activas, documentos, visitas e informes.

Propietario del software: JORGE ANTONIO CAMPOS CAMPOS

Objetivo de la capacitacion

Que cada usuario autorizado pueda ingresar al sistema, registrar solicitantes, revisar solicitudes, subir documentos, marcar VB, generar informes y mantener trazabilidad de su trabajo.

Regla principal de uso

Todo dato ingresado debe ser revisado antes de guardar. El sistema conserva documentos, VB y datos de ficha para proteger la informacion institucional.

13. Buenas practicas de uso

Antes de guardar

Revise cedula, nombre, programa, comite y documentos. Un dato correcto desde el inicio evita correcciones posteriores.

Documentos

Suba el archivo correcto en la pestaña correspondiente. No marque VB si falta el archivo o el dato solicitado.

Seguridad

Cierre sesion al terminar. Use siempre su usuario personal para que la auditoria sea correcta.

Resumen del flujo completo

Ingresar solicitante → asignar programa y comite → completar ficha → trabajar solicitudes activas → subir documentos → marcar VB → registrar visitas → generar informes → revisar auditoria cuando corresponda.
